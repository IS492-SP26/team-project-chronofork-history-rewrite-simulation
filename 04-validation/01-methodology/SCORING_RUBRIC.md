# Scoring Rubric

Each run is scored using the following dimensions.

Scores generally range from **1 to 5**:
- **1 = very weak**
- **3 = mixed / usable with major issues**
- **5 = strong / clearly useful**

---

**A. Historical Grounding:** Does the tool remain grounded in the historical situation?

**B. Causal Plausibility:** Does the tool explain why consequences follow from a divergence?

**C. Persona Fidelity:** Do historical actors remain distinct and believable?

**D. Narrative Coherence:** Does the interaction remain coherent across multiple turns?

**E. User-in-the-Loop Support:** Does the tool help the learner participate meaningfully without excessive prompting?

**F. Branch Handling:** Can the tool support checkpoint, branch, comparison, or backtracking logic?

**G. Reflection Quality:** Does the output help the learner reason about history?

**H. Safety / Plausibility Rail:** Can the tool reject absurd input or constrain implausible input productively?
